Седьмой блок кода отвечает за создание модального окна на странице.

1. let modal = document.getElementById("myModal"); - находит элемент модального окна по его id.
2. let btn = document.getElementById("myBtn"); - находит элемент кнопки, которая открывает модальное окно по ее id.
3. let span = document.getElementsByClassName("close")[0]; - находит элемент крестика, который закрывает модальное окно по его классу.
4. btn.onclick = () => {...} - добавляет обработчик события клика на кнопку, который открывает модальное окно.
5. span.onclick = () => {...} - добавляет обработчик события клика на крестик, который закрывает модальное окно.
6. window.onclick = (event) => {...} - добавляет обработчик события клика на любую область за пределами модального окна, который закрывает его.

Восьмой блок кода отвечает за отправку формы на странице.

1. let form = document.getElementById("myForm"); - находит элемент формы по его id.
2. form.addEventListener("submit", (event) => {...}) - добавляет обработчик события отправки формы, который будет вызываться при отправке формы.
3. let formData = new FormData(form); - создает новый объект FormData, который будет содержать данные из формы.
4. fetch("/path/to/server", {...}) - отправляет запрос на сервер с помощью метода fetch().
5. method: "POST", body: formData - определяет метод запроса и данные, которые будут отправлены на сервер.
6. event.preventDefault(); - предотвращает стандартное поведение браузера при отправке формы.
