Первый блок кода отвечает за переключение иконки меню и активацию/деактивацию навигационного меню при клике на иконку. 

1. let menuIcon = document.querySelector("#menu-icon"); - находит элемент иконки меню по его id.
2. let navbar = document.querySelector(".navbar"); - находит элемент навигационного меню по его классу.
3. menuIcon.onclick = () => {...} - добавляет обработчик события клика на иконку меню, который переключает классы CSS у иконки и навигационного меню, чтобы активировать или деактивировать его.

Второй блок кода отвечает за подсветку активной ссылки в навигационном меню при прокрутке страницы до соответствующей секции.

1. let sections = document.querySelectorAll("section"); - находит все секции на странице.
2. let navLinks = document.querySelectorAll("header nav a"); - находит все ссылки в навигационном меню.
3. window.onscroll = () => {...} - добавляет обработчик события прокрутки страницы, который проверяет, какая секция находится в верхней части экрана, и подсвечивает соответствующую ссылку в навигационном меню.
4. sections.forEach((sec) => {...}) - перебирает все секции на странице и для каждой определяет ее верхнюю позицию, высоту и id.
5. if (top >= offset && top < offset + height) {...} - проверяет, находится ли верхняя часть экрана в пределах текущей секции, и если да, то подсвечивает соответствующую ссылку в навигационном меню.

Третий блок кода отвечает за создание "липкого" навигационного меню при прокрутке страницы.

1. let header = document.querySelector("header"); - находит элемент заголовка страницы.
2. header.classList.toggle("sticky", window.scrollY > 100); - добавляет или удаляет класс CSS у заголовка, в зависимости от того, насколько прокручена страница.

Четвертый блок кода отвечает за анимацию элементов при прокрутке страницы.

1. ScrollReveal({...}) - создает новый объект ScrollReveal, который будет управлять анимацией элементов на странице.
2. ScrollReveal().reveal(...) - добавляет анимацию к указанным элементам на странице, используя параметры, такие как origin, distance, duration, delay и т.д.

Пятый блок кода отвечает за создание анимации набора текста при загрузке страницы.

1. const typed = new Typed(".multiple-text", {...}) - создает новый объект Typed, который будет управлять анимацией набора текста на странице.
2. strings: [...] - определяет массив строк, которые будут появляться при анимации.
3. typeSpeed: 100, backSpeed: 100, backDelay: 1000, loop: true - определяет параметры анимации, такие как скорость набора, скорость удаления, задержка перед удалением и повторение анимации.
